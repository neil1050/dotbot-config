# Dotbot configured configs

These configs are installed automatically by dotbot

## Pre-installation

* IMPORTANT: You must clone submodules before installing
    * If cloning, use the `--recurse-submodules` flag
    * If already cloned, use `git submodule update --init --recursive`

## Installation

1. cd into src
```
cd src
```

2. run install
```
./install
```
